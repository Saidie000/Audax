Getting started

NCOM Audax is a high-performance Windows drawing application built with Rust, Vello, and wgpu. Use this guide as the canonical reference for setup, tooling, and feature planning.

  • Clone the repository and install the Rust toolchain.
  • Build with cargo build --release.
  • Review module scaffolding in src/ for core systems.

Feature map

Canvas & Layers

Multi-layer canvas with blending, masks, and infinite mode.

Brush Engine

Pressure, tilt, jitter, spacing, and texture-driven strokes.

Tools

Selection, transform, QuickShape, snap-to-shape, and arrows.

Input

Surface Pen + Windows Ink integration with palm rejection.

MSI packaging

WiX tooling and scripts are included to produce a clean installer wizard. See installer/ for staging scripts, WiX templates, and assets placeholders.

  1. Build release: cargo build --release
  2. Stage output: installer/scripts/stage_release.sh
  3. Build MSI: wix build installer/wix/Product.wxs

Release workflow

The GitHub Actions workflow in .github/workflows/release.yml builds the MSI on Windows and uploads it to a GitHub Release. Tag a release with v0.1.0 to publish.
